An App of Ice and Fire

This is an application which demonstrates a segregated back-end and front-end. The back-end is a currently mocked, in this environment, with a front-end in AngularJS with Bootstrap 3 theme.

This application now brings new life to the process of development practice improvements, and is being used for demonstrating how to move an AngularJS 1.x application into using the Parcel.js bundler.

History

The original implementation of this application demonstrated how to build and deploy a more modern front-end application, either inside of an IBM Domino/XPages application container (NSF), or stand-alone with a mock-able back-end via JSON server. Additionally, it was deployed to a Bluemix IBM Cloud instance to demonstrate the power of bringing XPages to the cloud, and the development practices involved.

Application created with the love of the realm of Westeros, and of software development practices and tooling, by @edm00se.
